Stata How-To

Basics of Stata, as used in ECO372

  1. Get Started: get Stata running, organize your files, run your first do-file.
  2. Open and Save Data: open/import datasets from various formats, save/export datasets to various formats.
  • Explore and Describe Data: understand what is in the dataset, compute summary statistics
  • Clean and Modify your Data: create new variables, replace incorrect values
  • Basic Graphs: create basic univariate and bivariate graphs, customize appearance
  • Conditions, Subsetting Data: apply commands to a subset of observations according to some criteria. Dataset: CPS_2016.dta.
  • Test Hypotheses: test simple hypotheses in Stata, either using t-tests or regressions. Dataset: Blood_pressure_fictitious.dta.
  • OLS Regressions: run regressions and interpret Stata’s output
  • Instrumental Variables: run IV regressions using two-stage least squares (2SLS)
  • Export Regression Results using esttab: format your regression results in a nice table ready for inclusion in Word (or LaTeX).
